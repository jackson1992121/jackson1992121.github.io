Itinerary Booking System in Japan

This is the laravel project for itinerary website. The operators and agents make his own recommended itineraries via google map, and the customers can customize the routes of itinerary in consulting room with owners via Voxeet SDK.

Tech Stack Explanation

Laravel is used to build client and admin pages, and React is used for online meeting with Voxeet SDK, and express are used to socket communication. Database is MySQL.
