Cluble shopping site

This is the ruby on rails project for golf club shopping. The users can search for several golf clubs by category, maker, cost, status, name and so on. The site is fully responsive and it's also a mobile website.

Tech Stack Explanation

Ruby on rails is used to build client and admin pages, and Bootstrap CSS for the whole style and responsive. Database is PostgreSQL. CircleCI is used for continuous project test and deploy.
