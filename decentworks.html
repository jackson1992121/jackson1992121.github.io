Commute management system

This is the Laravel and react native project for controlling commutement. The user can set commuting time each day and then the time comes, app notifies to user, and automatically keep records of users. The users can also store their moods eveyday and the statistics of data can be analyzed by server.

Tech Stack Explanation

Laravel restful API is used for backend. And frontend is developed using react native. Database is PostgreSql.
